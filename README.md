#Raw-OS_Sqlite

---
Raw-OS_Sqlite is a [Sqlite3](http://www.sqlite.org/) porting on [Raw-OS](http://www.raw-os.org/) Real Time Operation System, with File System lay drive by [FatFs](http://elm-chan.org/fsw/ff/00index_e.html).
